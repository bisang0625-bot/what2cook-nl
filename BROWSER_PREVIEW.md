# 브라우저 미리보기 가이드

## Next.js 대시보드를 브라우저에서 보는 방법

### 1단계: 의존성 설치

```bash
# 프로젝트 루트 디렉토리에서 실행
npm install
```

### 2단계: 개발 서버 실행

```bash
npm run dev
```

실행 후 터미널에 다음과 같은 메시지가 나타납니다:
```
- ready started server on 0.0.0.0:3000, url: http://localhost:3000
- event compiled client and server successfully
```

### 3단계: 브라우저에서 접속

브라우저를 열고 다음 주소로 접속하세요:

**http://localhost:3000**

### 4단계: 대시보드 확인

브라우저에서 다음을 확인할 수 있습니다:

1. **헤더**: "K-Bonus" 로고와 "이번 주 마트 세일로 차리는 알뜰 밥상" 캐치프레이즈
2. **필터 바**:
   - 마트 선택: Albert Heijn (활성), Jumbo/Lidl (준비중)
   - 태그 필터: 👶 아이 식단, 🌶️ 매콤한 맛, 🥗 채식
3. **레시피 카드 그리드**: 세일 식재료 기반 한식 레시피 카드들
4. **카드 클릭**: 레시피 상세 정보 모달 (쇼핑 리스트, 절약 팁 포함)

### 주요 기능

- ✅ 반응형 디자인 (모바일/태블릿/데스크탑)
- ✅ 필터링 시스템 (마트 및 태그 필터)
- ✅ 레시피 상세 모달
- ✅ 네덜란드 오렌지 컬러 테마

### 개발 서버 중지

터미널에서 `Ctrl + C`를 눌러 개발 서버를 중지할 수 있습니다.

### 문제 해결

**포트 3000이 이미 사용 중인 경우:**
- Next.js가 자동으로 다른 포트(3001, 3002 등)를 사용합니다
- 터미널 메시지를 확인하여 실제 포트 번호를 확인하세요

**의존성 설치 오류:**
```bash
# 캐시 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install
```

**빌드 오류:**
```bash
# Next.js 캐시 삭제 후 재실행
rm -rf .next
npm run dev
```
